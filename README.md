# facthor-vikingoz

## Propósito del Repositorio

Este es un repositorio temporal diseñado para:

- **Creación de Issues**: Utilice este repositorio para reportar problemas, sugerencias y discusiones del proyecto.
- **Compartir Código Fuente**: El código fuente del proyecto será compartido aquí al finalizar el desarrollo.
- **Actualizaciones Oficiales**: Todas las actualizaciones oficiales y material importante serán publicados en este repositorio.
- **Documentación Wiki**: Este repositorio servirá como parte de la wiki final de documentación del proyecto.

## Documentación

La documentación completa está disponible a través de GitHub Pages. Visite la [página de documentación](https://droa6.github.io/facthor-vikingoz/) para más información.

## Contribuciones

Para contribuir al proyecto, por favor:

1. Cree un issue describiendo el problema o mejora
2. Espere la revisión del equipo
3. Siga las guías de contribución que serán publicadas próximamente
